# Mi Web - Franco Vilcahuaman Lazo

## Descripción
Este proyecto es una aplicación web básica desarrollada en PHP como parte de una práctica. Permite realizar operaciones matemáticas simples como la suma de dos números y el cálculo del factorial de un número.

## Tecnologías utilizadas
- PHP
- XAMPP (Apache)
- Git

## Estructura del proyecto
- index.php → Página principal con enlaces a las operaciones
- suma.php → Permite sumar dos números
- factorial.php → Calcula el factorial de un número
